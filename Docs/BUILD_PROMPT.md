# Stormin' Shane Weather App — Build Prompt

Build a premium, modern iPhone weather app called **Stormin' Shane Weather App**. The app is a personal weather brand and storm-tracking platform built around Shane, who creates video weather updates for her audience. This is not a generic weather app — it is a storm-chaser-grade tool with a creator dashboard baked in.

## Target Platform

- iOS native: Swift / SwiftUI
- iPhone 14 and newer
- iOS 17+
- Dark-mode first, light mode supported
- Optimized for one-handed use and quick glanceability during severe weather events

## Product Phases

1. MVP: Home/current conditions, hourly + 10-day forecast, RainViewer radar, saved locations, NWS alerts banner.
2. Full radar, tornado tracker, push notifications.
3. Shane Creator Mode: gated dashboard, uploads, publishing, public feed.
4. Chaser integrations, Critical Alerts entitlement, polish, TestFlight/App Store.

## Compliance Flags

- Critical Alerts require Apple approval.
- Use public APIs and proper attribution for chaser streams. Do not scrape.
- No third-party trackers without explicit privacy review.
