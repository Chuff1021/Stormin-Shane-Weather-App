# Stormin' Shane Weather App

A real installable weather web app/PWA built from the uploaded Stormin' Shane design starter.

## Run

```bash
npm install
npm run dev
```

## Features
- Live Open-Meteo forecast data
- NWS active alerts
- City search/geocoding
- Saved locations
- Fahrenheit/Celsius preference
- RainViewer radar map with timeline playback
- App-style iPhone PWA shell

The original uploaded design files are preserved in `reference/starter/`.
